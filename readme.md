uazui
===

## About 
uazuiで快適な環境を構築したい


## Try

```
bundle install --path vendor/bundle ## setup itamae
mv config.json.sample config.json 
editor config.json ## Anything you like
bundle exec itamae local uazui.rb -j config.json ## provisioning ( Reccomend: In screen )
```
